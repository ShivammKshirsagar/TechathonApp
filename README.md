# Loan Application

AI-powered loan application system built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `/app` - Next.js app router pages
- `/components` - React components organized by feature
- `/styles` - Global styles and Tailwind configuration
- `/providers` - Context providers (Theme, etc.)

## Features

- Chat-based loan application interface
- KYC verification
- Underwriting decision engine
- Salary slip upload and extraction
- Loan approval summary
- Dark mode support
